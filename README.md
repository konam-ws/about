# Konam

This repository is used to explain Konam.

Konam is a Spain-based software development company that leverages AI to build tools that help save people's time.

Currently, we are focused on creating Kubes, an application for simple project management. 
This application is currently free for anyone who signs up until a certain point in time. 
You can try it out for yourself, create an account, and claim your free-for-life spot at https://kubes.work.

We are also building Envia, a form submission collector that focuses on being an easy-to-use link between your forms and submissions, providing a few tools to help you, such as spam detection, organization, analytics, and more.
Envia is also free to use up to a certain point and then provides several pricing options.
You can try it out for yourself at https://envia.konam.dev.

Konam is still early in its journey, and we aim to provide many more applications that solve real problems and save people time.
